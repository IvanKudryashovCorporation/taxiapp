## Дополнительные команды, которые помогут получить 1-в-1

После того как Claude Code сделает первый экран:

> «Открой `Профсоюз Рассвет.html` в браузере, найди экран P-01 LoginScreen, открой DevTools, измерь точные размеры заголовка "Доброе утро, Севастополь.", input телефона, кнопки. Сравни с твоей реализацией построчно — что не совпадает? Перечисли все различия списком и поправь.»

Если получается «похоже, но не точно»:

> «Отличия от эталона недопустимы. Открой DevTools на эталоне для конкретного элемента, скопируй computed styles (font-size, line-height, padding, margin, border-radius, background), и приведи свою реализацию к ним 1-в-1.»

Если Claude Code предлагает упростить или улучшить:

> «Я не просил улучшать дизайн. Точно скопируй то, что в эталоне. Если эталон выглядит неоптимально — это ОК, дизайн уже согласован. Твоя работа — реализовать его точно, не редизайнить.»
